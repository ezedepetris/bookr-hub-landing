Ejemplos de Flujo de Reservas

Mira cómo diferentes negocios personalizan su flujo de reservas:

Flujo de Barbería

  1. Selecciona servicio (Corte, Corte + Barba, Solo Barba)
  2. Elige barbero (o "Primero Disponible")
  3. Pick fecha y hora
  4. Ingresa nombre y teléfono
  5. Paga $10 de depósito (opcional)
  6. Recibe confirmación

Flujo de Salón de Belleza

  1. Explora categorías de servicios (Corte, Color, Tratamientos)
  2. Selecciona servicio específico (Balayage, $220)
  3. Elige estilista (o "Cualquier Disponible")
  4. Ver fechas disponibles (próximas 2 semanas)
  5. Selecciona horario
  6. Crea cuenta o continúa como invitado
  7. Añade notas para el estilista
  8. Confirma reserva

Flujo de Spa

  1. Selecciona tipo de tratamiento (Masaje, Facial, Cuerpo)
  2. Elige tratamiento específico (60-min Masaje Profundo, $130)
  3. Selecciona preferencia de terapeuta
  4. Pick fecha del calendario
  5. Elige horario (mañana, tarde, noche)
  6. Ingresa datos del cliente
  7. Añade solicitudes especiales
  8. Revisa y confirma
  9. Paga 30% de depósito

Preguntas Frecuentes

¿Puedo personalizar mi página de reservas para que coincida con mi marca?
Absolutamente. BookrHub ofrece personalización completa de tu página de reservas, incluyendo tu logo, colores de marca, fuentes e imágenes. También puedes elegir entre docenas de plantillas profesionales diseñadas para diferentes tipos de negocios, o trabajar con nuestro equipo para crear una página de aterrizaje completamente personalizada.
¿Ofrecen plantillas de páginas de aterrizaje?
¡Sí! Tenemos docenas de plantillas diseñadas profesionalmente listas para usar. Simplemente elige una plantilla, personaliza colores y marca, añade tus servicios y estás listo. Todas las plantillas están optimizadas para móviles y amigables con SEO. Si necesitas algo único, nuestro equipo puede ayudarte a crear una página de aterrizaje completamente personalizada.
¿Puedo personalizar cómo funciona mi flujo de reservas?
Sí. BookrHub te permite personalizar cada paso de tu flujo de reservas. Elige qué servicios aparecen, cómo los clientes seleccionan personal, qué información recopilas, qué opciones de pago están disponibles, y más. Ya sea que necesites un flujo simple de 3 pasos o un proceso complejo de reserva multi-servicio, lo tenemos cubierto.
¿Necesito mi propio sitio web para usar BookrHub?
No. Tu página de reservas de BookrHub sirve como tu presencia online. Incluye tus servicios, personal, precios, fotos y funcionalidad de reservas, todo en un solo lugar. Puedes añadir este enlace a tu bio de Instagram, página de Facebook, o cualquier sitio web existente. Muchos de nuestros clientes usan su página de BookrHub como su sitio web principal.
¿Puedo usar mi propio nombre de dominio para las reservas?
Sí. Mientras obtienes una URL gratis (tunegocio.bookrhub.com), puedes actualizar para usar tu propio dominio personalizado (book.tunegocio.com o reservas.tunegocio.com). Esto se ve más profesional y facilita que los clientes encuentren y recuerden tu página de reservas.
¿Cómo funciona el sistema de depósitos?
Puedes requerir que los clientes paguen un depósito al reservar, ya sea un porcentaje del precio del servicio o un monto fijo. Los depósitos se acreditan hacia el servicio y se pierden si el cliente no se presenta. Esto reduce las inasistencias un 60-80% y protege tu tiempo. Tú estableces las reglas de depósito, y BookrHub maneja todo automáticamente.
¿Qué métodos de pago puedo aceptar?
BookrHub soporta múltiples métodos de pago incluyendo tarjetas de crédito/débito, métodos de pago locales, MercadoPago (para LATAM), Stripe, y más. Puedes ofrecer pago completo, solo depósitos, opciones de pagar en el local, o pagos divididos. Todo el procesamiento de pagos se maneja de forma segura a través de nuestros socios de pago integrados.
¿Cobran comisión en las reservas?
No. BookrHub cobra una suscripción mensual fija con cero comisión en reservas. A diferencia de plataformas como Fresha que toman 5-8% de cada reserva, tú te quedas con el 100% de lo que ganas. Las únicas tarifas son los costos estándar de procesamiento de pagos.
¿Puedo gestionar múltiples miembros del personal?
Sí. BookrHub incluye gestión integral del personal: calendarios individuales, cualificaciones de servicio, horarios de trabajo, solicitudes de tiempo libre, permisos basados en roles, y seguimiento de rendimiento. Cada miembro del personal puede tener su propio inicio de sesión y página de reservas, o puedes gestionar todo bajo una cuenta.
¿Y si necesito ayuda con la configuración?
Ofrecemos asistencia de configuración para ayudarte a comenzar. Nuestro equipo puede ayudar a configurar tus servicios, personalizar tu flujo de reservas, diseñar tu página de aterrizaje, y más. Consulta tu plan para el soporte de configuración incluido, o contáctanos sobre servicios premium de onboarding.
